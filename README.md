# Simple Music Player 

Welcome to the **Simple Music Player**! This is a basic web music player application built with JavaScript, HTML, and CSS. It allows you to play, pause, skip, and control the playback of audio files, giving a lightweight and straightforward experience for listening to your favorite tracks.

## 🚀 Features
- **Play & Pause**: Easily start and stop audio playback.
- **Skip Tracks**: Jump to the next or previous track in the playlist.
- **Progress Bar**: Visualize the track’s progress and click to skip.
- **Volume Control**: Adjust the playback volume.
- **Responsive Design**: Works on both desktop and mobile browsers.

## 🛠️ Technologies Used
- **JavaScript (ES6+)**: Core functionality and DOM manipulation
- **HTML5 Audio API**: For managing audio playback
- **CSS3**: Styling and layout
- **FontAwesome**: Icons for player controls
